Variaveis

variaveis sao muitos importantes em qualquer linguagem de programacao tanto como em python!

É um nome que é reservado um espaco pelo computador para receber dados! 

      
      var = 123
      
      var2 = "ola"
      var3 = 12,4

Como visto o nome var e atribuido o valor 123 com o simbolo de igual. Percebeu que os dados para cada variavel sao diferentes vamos Aprender os timpos de Variaveis 

          
        
 var = 123 → variavel numerica
 var1 = "ola" → variavel de caracter
 var2 = 12.4 → variavel numerica decimal
